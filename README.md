# lego-ui-admin-pro

基于 [lego-ui](http://172.22.1.88/fe/lego-ui) 的中后台项目管理系统。

## 开发 

环境依赖: node `v8.15.0` 及以上，nvm `v0.33.1` 及以上

clone 项目： 

```bash
git clone 

cd lego-ui-admin-pro

yarn
```

运行：

```bash
yarn dev
```
